Descripcion

"Smart Identifier" es una solución de control de acceso avanzada que utiliza tecnología de última generación para garantizar la agilización de procesos y el control de acceso a las instalaciones educativas que lo soliciten. El sistema utiliza un software de reconocimiento por medio de NFC y de RFID para autenticar la identidad de las personas que acceden a la escuela. Además, el sistema cuenta con características avanzadas de monitoreo y seguimiento de la asistencia de los estudiantes, y puede integrarse con otros sistemas de registro, como cámaras o inclusive lector de huellas dactilares

Objetivos

Objetivo general:

Diseñar un prototipo para automatizar el acceso de alumnos en las aulas de ITSOEH, utilizando tarjetas NFC con tecnologias Wifi, ESP32-CAM, LoRaWAN, node-red y MQTT.

Objetivos especificos:

  • Analizar las configuraciones y las formas de comunicación del sensor PN532.
  • Desarrollar un programa en la plataforma de Arduino para la lectura del sensor y envío de datos por puerto serial.
  • Establecer comunicación entre el lenguaje de programación C# y el puerto serial.
  • Diseñar y desarrollar una base de datos acorde a las necesidades de este.
  • Realizar y establecer conexión con la base de datos.
  • Desarrollar e implementar la lógica del sistema.
  • Diseño y desarrollo de interfaces de sistema.
  • Desarrollar un programa en la plataforma de Arduino para la lógica de la ESP32-CAM.
  • Diseñar y configurar las herramientas de Node-RED.
  • Montar y configurar un Broker de MQTT y los dispositivos que serán clientes y publicadores.
  • Desarrollar un programa en la plataforma de Arduino para la lógica de la ESP32 con LoRaWAN.
  • Evaluar temas y datos enviados por MQTT.

Justificacion

Con base al planteamiento del problema se propone el desarrollo de un prototipo de un dispositivo que pueda leer datos escritos en tarjetas o tecnologías NFC por medio de un sensor PN532 el cual enviará el dato con ayuda de Arduino, a través de la comunicación serial, recibida por un Raspberry pi 3 B. Por medio de una interfaz gráfica desarrollada en el lenguaje de programación c# se leerá los datos de la tarjeta NFC obtenidos de la comunicación serial; mismo que podrán manipularse por medio de una base de datos implementada en un servidor, estos datos serán los datos del estudiante foto, nombre, matricula y materia.

Este prototipo dará solución al proceso de registro de asistencias en cual es repetitivo, con lleva un tiempo, usualmente no hay un sistema óptimo para realizarlo el cual los profesores pueda ocupar, así mismo en ciertas ocasiones con el sistema actual que llevan los profesores para realizar esta tarea suele tener problemas como que no se realice el registro, desorganización por la cantidad de salones a los que les dan clases o la perdida de información.

Para evitar que estas situaciones sucedan, el prototipo conforme a las características que tiene podrá dar solución a este tipo de situaciones al tener el registro en una base de datos local dentro de a institución. Y al mismo tiempo permitirá que la información se fácilmente consultada por los usuarios que requieran la información como pueden ser profesores, jefes de tutorados o el jefe de carrera.
